## 主题SEO优势

搜索引擎优化（SEO）是 Mirai 未来主题的核心优势之一。内置全套增强方案，覆盖结构化数据、Meta 标签、索引推送、链接规范化、站点地图四大维度，帮助网站在百度、谷歌、必应等搜索引擎中获得更好的收录与排名。

### 全自动 Schema 结构化数据

自动生成符合 Schema.org 标准的 JSON-LD 标记，共支持 **5 种 Schema 类型**：

- **WebSite / Organization**：首页站点信息结构化
- **Article**：文章页富摘要展示（标题/作者/发布时间/封面图）
- **BreadcrumbList**：面包屑导航层级标记
- **CollectionPage**：分类页与标签归档页
- **SearchResults**：搜索结果页

覆盖全部页面类型，显著提升搜索引擎结果页（SERP）的点击率与专业展现形式。

### 智能 Meta 标签优化

针对不同搜索引擎算法自动优化 Title、Description 与 Keywords 标签，同时内置以下能力：

- **Open Graph 协议**：社交平台分享时自动提取标题、描述、封面图
- **Twitter Card**：Twitter/X 平台分享卡片优化
- **文章级 SEO 字段**：每篇文章可独立设置 SEO 关键词、SEO 描述、自定义封面图
- **Canonical 规范化**：自动输出规范链接，防止重复内容导致权重分散
- **外链 nofollow**：站外链接自动添加 nofollow 属性，保护网站权重不流失

同步移除 pingback 与 xmlrpc/wlw 等冗余请求，减少无效爬虫干扰。

### 即时索引推送

集成两大即时索引通道，确保新发布内容在几秒内被搜索引擎抓取：

- **百度主动推送**：文章发布后自动推送到百度站长平台
- **IndexNow 协议**：一键推送至必应（Bing）、Yandex、Seznam 等支持该协议的搜索引擎

此外还提供 RSS 2.0 与 Atom 双格式订阅输出，付费阅读文章自动过滤隐藏内容，保障订阅用户体验。

### 自动站点地图（Sitemap）

内置完整的 Sitemap 自动生成与管理功能：

- **自动更新**：文章发布、更新、删除时自动更新 Sitemap
- **多格式支持**：生成标准 XML 格式的 Sitemap 文件
- **分片索引**：支持大量内容的 Sitemap 索引文件，确保搜索引擎高效抓取
- **后台可视化配置**：可自定义更新频率、优先级等参数
- **独立管理面板**：后台提供 Sitemap 状态查看与手动生成入口

## 性能优化与响应式设计

页面加载速度是用户体验的核心指标，Mirai 未来主题从图片处理、资源加载、代码执行三个层面进行全方位性能优化。

### 图片懒加载与现代格式

采用 lazysizes 智能懒加载技术，配合原生 `<picture>` 响应式标签与 `srcset` 自适应属性，原生支持 WebP 与 AVIF 现代图片格式，在不损失画质的前提下大幅减少传输体积。**代码高亮组件仅在文章包含代码块时按需加载**，避免无关页面引入冗余资源。

### 预加载与异步执行

- **DNS Prefetch 预解析**：预解析外部域名，减少 DNS 查询延迟
- **Instant Page 预加载**：鼠标悬停链接时预加载目标页面，实现"点击即打开"
- **脚本异步加载**：所有 JavaScript 采用异步方式加载，非关键资源延迟执行
- **关键 CSS 内联**：首屏渲染关键样式内联到 HTML，其余按需加载

引入 RemixIcon 图标字体替代内联 SVG，图标统一管理更轻量。内置多级缓存机制减少重复计算开销。

## 付费阅读与 VIP 会员体系

提供从内容定价到资金流转的完整闭环，是目前 Typecho 生态中商业化程度最高的主题方案之一。

### 灵活的付费阅读模式

支持两种付费模式以适配不同的内容分发场景：

| 模式         | 说明                                     | 适用场景                |
| ------------ | ---------------------------------------- | ----------------------- |
| 整篇付费     | 全文锁定，购买后完整阅读                 | 付费课程、深度报告      |
| 部分隐藏付费 | 用 `[pay]...[/pay]` 标签包裹需付费段落 | 小说连载、混合免费+付费 |

每篇文章可在后台独立设置价格、购买方式限制（在线支付/余额支付/仅在线/仅余额）、是否强制登录、作者分成比例等参数。

### 多渠道支付接入

支持 **5 种支付网关 + 余额支付 + 积分兑换**，覆盖几乎所有支付场景：

| 支付网关       | 说明                                                |
| -------------- | --------------------------------------------------- |
| 易支付         | 聚合支付平台，支持微信/支付宝/QQ，兼容 V2（RSA）和 V1（密钥）两种接口 |
| 虎皮椒支付     | 聚合支付平台，分别配置微信和支付宝渠道              |
| 支付宝官方支付 | 支付宝开放平台官方接口（RSA 密钥对认证）            |
| 微信官方支付   | 微信支付 V3 API（商户号 + APIv3 密钥 + 证书序列号） |
| 支付宝当面付   | 支付宝官方扫码支付（生成二维码，适合移动端）        |
| 余额支付       | 用户充值余额后直接抵扣                              |
| 积分兑换       | 使用积分抵扣订单金额                                |

用户端支持微信、支付宝、QQ 三种支付渠道，后台可按渠道独立选择使用的支付网关（如微信渠道可选易支付、虎皮椒或微信官方）。

**支付安全与风控：**

- 并发锁机制：数据库锁 + 文件锁双重保障，防止重复支付
- 订单超时：默认 30 分钟自动关闭（后台可配置）
- 访客支付：支持未登录用户通过 Cookie 令牌完成支付
- 异步通知：所有支付网关均支持异步回调，确保订单状态准确同步

### 三级 VIP 会员等级系统

Mirai 未来主题内置完整的 **三级会员体系**（一级/二级/三级），是付费阅读之外的第二大变现引擎：

- **5 种时长套餐**：月度（30天）/ 季度（90天）/ 半年（180天）/ 年度（365天）/ 永久（不过期），每种时长独立定价
- **4 种折扣模式**：每篇文章可单独选择——沿用全局折扣比例 / 自定义各等级价格 / 会员免费阅读 / 不参与会员折扣
- **灵活升级机制**：低等级会员升级高等级时自动计算补差价；续费支持原价续费 / 续费打折 / 续费固定价 / 不允许续费四种模式
- **自定义命名**：三级会员名称均可后台自定义，默认为一级会员/二级会员/三级会员

创作者激励方面，支持全局默认分成比例与单篇文章单独覆盖，包含实时收益统计与提现申请功能（可配置最小提现额与手续费）。

## 积分系统与推荐返利

除了付费阅读和 VIP 会员，Mirai 未来主题还提供完整的积分生态和推广返佣机制，构建活跃的用户社区。

### 积分系统

内置完整的积分获取与消费体系：

- **多场景积分获取**：注册奖励、签到奖励、评论奖励、被点赞奖励、购买 VIP 奖励、使用邀请码奖励
- **积分消费场景**：积分兑换余额、积分购买文章、积分参与抽奖
- **后台积分管理**：支持批量调整用户积分、查看积分变动记录、设置积分规则
- **积分统计面板**：实时查看全站积分流通数据

### 推荐返利系统

内置推广返佣机制，激励用户主动推广内容：

- **专属推广链接**：每个用户拥有独立的推广链接（`?ref=用户ID`）
- **多级返佣**：支持配置推广层级关系，上级可获得下级消费的一定比例返佣
- **返佣记录查询**：用户可在个人中心查看推广人数、返佣明细

### 邀请码系统

完整的邀请码管理功能，用于控制注册入口并激励用户邀请：

- **批量生成**：后台支持批量生成邀请码，可设置积分奖励、余额奖励、VIP 等级奖励
- **邀请码列表管理**：查看邀请码状态（未使用/已使用/已过期）、使用人信息
- **多格式导出**：支持 TXT 和 CSV 两种格式导出，可选择仅导出邀请码或全部字段
- **一次性消耗**：邀请码为一次性使用，注册后自动标记为已消耗
- **奖励自动发放**：使用邀请码注册后，邀请人和被邀请人均可获得预设奖励

## 第三方社交登录

内置强大的第三方登录系统，支持 **16 种社交平台**一键登录，大幅降低用户注册门槛。

### 两种接入模式

| 模式           | 说明                                                                 | 支持平台                                                       |
| -------------- | -------------------------------------------------------------------- | -------------------------------------------------------------- |
| 直接 OAuth     | 各平台独立配置 AppID / AppKey，直接对接官方 API                      | QQ、微信、微博、GitHub、Gitee、百度、支付宝（7 种）            |
| 彩虹聚合登录   | 通过彩虹聚合登录 API 统一接入，一次配置即可启用多个平台              | 在 OAuth 基础上扩展：钉钉、华为、小米、抖音、飞书、Google、Microsoft、Facebook、Twitter等多种方式 |

两种模式可共存，直接 OAuth 优先级高于彩虹聚合登录——同一平台若已配置直接 OAuth，则不会走聚合通道。

### 核心功能

- **一键登录 / 注册**：用户通过社交平台授权即可登录，首次登录自动创建账号并同步昵称与头像
- **账号绑定 / 解绑**：已登录用户可在个人中心绑定或解绑社交账号，同一类型仅允许绑定一个
- **防冲突机制**：同一社交账号不可绑定多个站点用户，避免账号混乱
- **头像自动同步**：首次绑定时自动同步社交平台头像（用户无头像时）
- **推荐人追踪**：第三方登录完整支持推荐链接（`?ref=`），确保推广关系不丢失

## 安装教程

### 运行环境要求

- **服务器环境**：Nginx（推荐），其他环境可自行尝试
- **PHP 版本**：PHP 7.4+，强烈推荐 PHP 8.3 或 8.4
- **PHP 扩展**：安装 opcache、imagemagick 扩展
- **数据库**：MySQL 5.7+
- **Typecho 版本**：1.3.0+
- **依赖插件**：MiraiCore 核心插件（必须安装并启用，否则主题无法运行）

**重要**：主题版本必须和插件版本一致，不一致可能会报错

### 安装步骤

1. 将 `Mirai` 文件夹上传至 Typecho 目录下的 `usr/themes/`
2. 将 `MiraiCore` 文件夹上传至 Typecho 目录下的 `usr/plugins/`
3. 登录后台 → 控制台 → 插件 → 启用「MiraiCore」
4. 进入控制台 → 外观 → 启用「Mirai」主题
5. 点击主题名称进入「设置外观」完成配置

### Nginx 伪静态配置

后台需启用地址重写功能，Nginx 伪静态规则如下：

```nginx
if (!-e $request_filename) {
        rewrite ^(.*)$ /index.php$1 last;
    }
```

保存后重启 Nginx，再到后台「永久链接」保存一次即可生效。
